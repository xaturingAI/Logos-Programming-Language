# Logos Programming Language Examples

This directory contains example code files demonstrating various features of the Logos programming language.

## Example Files

1. **hello_world.logos** - Basic "Hello, World!" program
2. **variables_and_types.logos** - Variables, data types, constants, and pattern binding
3. **functions.logos** - Functions, lambdas, higher-order functions, and generics
4. **classes_objects.logos** - Classes, objects, inheritance, and methods
5. **enums_pattern_matching.logos** - Enums, pattern matching with guards
6. **control_flow.logos** - Advanced control flow, pipeline operator, exception handling
7. **collections.logos** - Arrays, maps, sets, and tuple destructuring
8. **concurrency.logos** - Async/await and actor model concurrency
9. **advanced_types.logos** - Dependent types, linear types, and effect system
10. **multilang_integration.logos** - Multi-language integration with @python, @js, @rust annotations
11. **comprehensive_example.logos** - A complete example combining multiple features

## Running Examples

To run any of these examples, use the Logos compiler:

```bash
cargo run -- run examples/hello_world.logos
```

Or build and run:

```bash
cargo run -- build examples/hello_world.logos
./hello_world
```

Each example demonstrates specific features of the Logos language as described in the README.md file in the main directory.
